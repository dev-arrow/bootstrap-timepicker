Timepicker A simple widget to easily select a time for an input element

About timepicker

A streamlined, but flexible, take on the traditional javascript timepicker plugin with only the minimum required functionality and smart defaults.

Using bootstrap-timepicker

Call the timepicker via javascript:

$('.timepicker').timepicker();

Options

Name type default description
step integer 1 Specify a step for the minute field.
defaultTime string 'current' Set the initial time value. Setting it to 'current' sets it to the current time.
